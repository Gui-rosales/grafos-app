## Iniciando

Setando o projeto localmente.

### Pré-requisitos

Docker
Python

### Instalação

1. Clone o repo e entre dentro da pasta

```sh
git clone https://github.com/wilsntn/fast-api-backend.git
```

2. Crie um virtual environment

```sh
python -m venv venv
```

3. Monte os containers

```sh
docker compose up --build
```

4. Pronto!
